Examples

For short sequences
./wavefront -P data/NC_000898.1.fasta -T data/NC_007605.1.fasta -k -1 -DP2

./wavefront -P data/NC_000898.1.fasta -T data/NC_007605.1.fasta -k -1 -DP2 (it takes 116 seconds)
